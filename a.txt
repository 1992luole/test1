一. 第一次安装 
设置name 和email
git config --global user.name "luo"
git config --global user.email "****@**.com"

二. 在项目根目录下打开 git软件
初始化 
git init

三. 将文件放在 暂存区
git add 路径(./a.index)

当前文件下全部修改或新增的文件一次性放入
git add ./回车

四. 将文件从暂存区放入 版本库
git commit -m "备注一定要写"

如果没有写 -m "" 会进入一个编辑窗口 退出
esc键:q!	或关闭窗口重启窗口

直接将文件从工作区放入版本库
git commit --all -m "总备注"

五.查看状态
git status
  modified: a.text  
红色在工作区
绿色在暂存区
工作区干净 working directory clean 是在版本库

六.清屏 clear 
上下箭头复制代码

七.忽略文件
在项目根目录下建一个文件 .gitignore
在这个文件里
  /.idea       忽略.idea文件
  /.gitignore  忽略 自己
  /js          忽略js文件夹
  /js/*.js     忽略js文件夹下的所有.js文件

八.日志 打印提交的版本和时间
git log 历史提交日志  按q 退出
git log --oneline  简洁版

九.版本回退
git reset --hard Head~0  0代表回到上一次版本 ^^代表上一个两个是上上一次
1 代码回到上上次
2 git reset --hard 版本号

打印所有的版本操作 
get reflog 
回到任意一步的版本
git reset --hard 版本号

十.创建分支
1.git branch 分支名   没有分支名就是查找所有分支
2.切换到
git checkout 分支名
3.正常写代码 提交 
4.切换到主分支
5.合并分支 
git merge 分支名    updating 版本号  说明成功

合并冲突
另个分支同时修改相同代码
